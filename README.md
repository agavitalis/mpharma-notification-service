## Description
This project is the notification service of mPharma application

## Prerequisites
- Node.js 
- Yarn or NPM
- MongoDB
- RabbitMQ


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run build
$ npm run start
```
## To access
Open the browser and go to http://localhost:5002 to access

## Test

```bash
# unit tests 
$ npm run test

```
